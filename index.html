leaflet built in

MapBox.js is built on top of Leaflet — an open-source mapping library. Leaflet's API handles much of the fundamental operations of using maps. MapBox.js API documentation should be read in conjunction with the Leaflet API reference.

Browser Supported

MapBox.js aims to be compatible with IE8+ and all modern browsers.

Open specifications

The library takes advantage of open specifications, including TileJSON and UTFGrid.

The markers API uses GeoJSON, a simple & open standard for geo data. It's based on JSON and supports features like points and polygons.

Optimized for mobile

MapBox.js is optimized for mobile devices and small screens by default. For additional tools for managing mobile maps, checkout the Mobile section in the documentation

Get started fast

MapBox.js is hosted on a worldwide content-distribution network plus there's no API key to use it. All that's needed is a reference to the script & CSS in your HTML document
